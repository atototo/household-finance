# 💰 household-finance

부부 가계부 Notion 관리 플러그인 for Claude Code.

디스코드, 터미널 등에서 자연어로 가계부를 관리합니다.

## 기능

- **월별 예산 관리**: 월급 입력 → 고정지출 자동차감 → 잔여예산(=남은 돈) 실시간 표시
- **수입/지출 기록**: "오늘 점심 12000원 카드" → 자동 파싱, 카테고리 분류, Notion 기록
- **빚 상환 추적**: 마이너스통장, 전세대출, 카드할부 등 종류별 자동 처리
- **저축 목표 관리**: 적금 입금 기록, 진행률 자동 계산
- **투자 매매 기록**: 주식 매수/매도/배당
- **고정지출 관리**: 적용기간 설정으로 할부 등 기한부 항목 자동 ON/OFF
- **카드 발생주의**: 긁는 순간 지출 반영, 카드대금 납부일 이중차감 방지

## 입력 방식

- 자연어: "오늘 점심 12000원 카드"
- 문자 복사: 카드 결제 알림 문자 그대로 붙여넣기
- 스크린샷: 카드/은행 앱 알림 이미지
- 은행 PDF: 거래내역 PDF 업로드

## 예산 구조

```
잔여예산 = 월급 - 고정지출비 - 실제지출 - 상환금액 - 저축금액
```

모든 출금 경로(지출/상환/저축)가 월별 예산에 자동 합산되어 **진짜 남은 돈**이 보입니다.

## 설치

```bash
claude plugin add atototo/household-finance
```

## 사전 준비

1. Notion에 가계부 DB 세팅 필요 (별도 템플릿 제공 예정)
2. Claude Code에 Notion MCP 연결 (플러그인 또는 Integration 토큰)
3. `references/notion-schema.md`의 DB ID를 본인 Notion DB에 맞게 수정

## 사용 예시

```
> 오늘 점심 12000원 카드
✅ 수입/지출 기록 완료 (식비, 카드, 12,000원)

> 이번 달 월급 350만원
✅ 4월 남편 예산 월급 업데이트: ₩3,500,000

> 예산 얼마 남았어?
📊 4월 남편: 잔여예산 1,245,000원
📊 4월 아내: 잔여예산 580,000원

> 마이너스통장 50만원 갚았어
✅ 상환 기록 완료 (마이너스통장, 500,000원)

> 5월 예산 만들어줘
✅ 5월 남편/아내 예산 생성, 고정지출 12건 자동 연결
```

## Notion DB 구조 (8개)

| DB | 용도 |
|---|---|
| 월별 예산 | 월급 → 잔여예산 자동계산 |
| 고정지출 항목 | 매달 반복 고정지출 마스터 |
| 수입/지출 | 일상 수입/지출 기록 |
| 빚 정리 | 빚 목록, 잔액 자동계산 |
| 상환 기록 | 빚 상환/인출 기록 |
| 저축 목표 | 저축 목표, 진행률 자동계산 |
| 저축 기록 | 저축 입금/인출 기록 |
| 투자 관리 | 주식 매매 기록 |
